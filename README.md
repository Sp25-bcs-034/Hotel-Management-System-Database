# Hotel Management System Database

A comprehensive Hotel Management System developed as a semester project for the Database Systems course at COMSATS University Islamabad.

The project demonstrates the complete database development lifecycle, including database design, SQL implementation, MongoDB collections, CRUD operations, joins, and aggregation queries.

---

## Overview

This project models the operations of a modern hotel management system using both relational and NoSQL database concepts.

The system supports:

- Hotel Management
- Room Management
- Guest Management
- Reservations
- Billing
- Employee Management
- Housekeeping
- Room Availability

The project combines database design with practical implementation in Oracle SQL and MongoDB.
